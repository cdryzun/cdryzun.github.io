    • 解决 Windows git pull 卡住不动问题

      • 原因 在 git clone 时后面仓库地址是非公开的 需要验证用户及密码 而windows 默认是已 弹窗形式

      • 解决方法：切换模式

        1

        git config --system --unset credential.helper

    • 解决 git 每次 pull/push 多需要输入账号密码问题

      1

      git config --global credential.helper store
